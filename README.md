# VPN-Connector

Simples script de conexão VPN que utiliza os serviços VPNBook e FreeVPN.

## Requerimentos

 - openvpn

## Instalação
    # ./installer.sh
  
## Uso
    # vpn-connect

### Aviso
 Renomear os arquivos .ovpn acarretará em erros no script.

## Serviços Suportados
 - VPNBook 
 - FreeVPN
